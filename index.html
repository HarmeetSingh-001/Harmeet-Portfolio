Harmeet's Top Games

This project was designed to practice using HTML, CSS, JS and database connections to make a e-commerce site. This one utilized PHP for all backend data management. VibeZ

This project was designed to practice using everything involving web development. HTML, CSS, JS, API calls, and database mangement. This was also a group project. My part of this project consists of the front end functions as well as api calls to our choice of api "Deezer" to fetch songs. Then sending them to our custom database through MongoDb The idea of this website was to create an app similar to letterbox to rate songs you listen to and find new songs through a sort of speed dating for songs section. From the home page you want to head down to the login or sign-up page. The authentication is not too secure so do not truly use a real password or username. Once you login you will be directed to the true site where you can search for songs by genre, however you can not search with a search bar since we do not want the users to be able to find whatever song they like but rather experience songs they might not have before. Speed Search is your speed dating app. Like or dislike songs and it will continue to find you new songs you might enjoy. You will also rate the song when you like it and then can remove or update it in My Songs tab.
